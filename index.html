📖 How to Use This App

1. Select Medium: Choose between Water/Glycol mix or Air.

2. Select System Type:
• Batch: For closed tanks, storage vessels, or room air volumes.
• Flow Rate (L/s): For inline heaters, coils, boilers, or pump loops delivering continuous Litres per second.

3. Select What to Solve:
• Batch Mode: Calculate time required (seconds/hours) OR power required (kW).
• Flow Mode: Calculate power required (kW) for a given flow OR temperature rise ($\Delta T$) produced by a heater.
